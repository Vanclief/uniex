## Development

### Testing

Since multiple endpoints are private, you need an API Key for each exchange test suite you want to run.

Current format is 

`EXCHANGENAME_API_KEY` for the API Key

`EXCHANGENAME_SECRET_KEY` for the Secret Key


**Run tests**

`make test`
